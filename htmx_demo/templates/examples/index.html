jQuery/AJAX vs HTMX Training Examples

About These Examples

This demo application showcases common web interaction patterns implemented in both jQuery/AJAX (traditional approach) and HTMX (modern, declarative approach).

Each example is fully functional and can be interacted with to see how the different approaches work in practice.

Patterns Demonstrated

1. Form Submissions

Learn how to handle form submissions with inline validation and success messages. See the difference between manual AJAX calls and HTMX's declarative attributes.

2. Live Search/Filtering

Implement real-time search functionality. Compare manual DOM manipulation with HTMX's automatic content replacement.

3. Infinite Scroll/Lazy Loading

Create infinite scroll pagination. See how HTMX simplifies loading more content as users scroll down the page.

4. Modal Dialogs

Open modal dialogs with server-rendered content. Compare manual Bootstrap modal management with HTMX's streamlined approach.

5. Dynamic List Operations

Build a todo list with add/remove/toggle functionality. See how HTMX reduces JavaScript complexity for common CRUD operations.

6. Dependent Dropdowns

Implement cascading select dropdowns (Country → State → City). Compare manual event handling with HTMX's automatic approach.

7. Polling/Auto-refresh

Create a live status dashboard with automatic polling. See how HTMX eliminates the need for setInterval and manual state management.

🚀 SETUP (3 STEPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Run Migration:
   $ docker-compose -f docker-compose.local.yml run --rm django python manage.py migrate

2. Create Sample Data:
   $ docker-compose -f docker-compose.local.yml run --rm django python manage.py create_sample_data

3. Start Server:
   $ docker-compose -f docker-compose.local.yml up

🔍 COMPARISON HIGHLIGHTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

jQuery/AJAX (~80 lines):
  • Manual AJAX calls with $.ajax()
  • JSON response parsing
  • Manual marker management
  • Complex event coordination
  • Client-side HTML building

HTMX (~60 lines):
  • Declarative hx-* attributes
  • HTML response (no JSON parsing)
  • Event-driven updates
  • Automatic form coordination
  • Server-side HTML generation

Result: 25% less code + cleaner architecture
